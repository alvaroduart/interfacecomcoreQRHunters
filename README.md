# COMO TESTAR A APLICAÇÃO

  ## Login
  - Realize o cadastro e faça o login

  ## Como testar a leitura de QrCode? 
  - Na tela principal (câmera), leia os QrCode's presentes na pasta *QrCodes*
  - O QrCode com nome "Errado" está com a localização errada, logo, não será validado (não irá aparecer na lista de leitura)
  - O QrCode com nome "Correto" está com a localização de sua casa (latitude e longitude). Independente de sua escolha (errada ou correta), o ponto irá aparecer na lista de leitura.
